# CSE3311
~~~
This is the repository of Team 4 of CSE 3311 - Object-Oriented Software Engineering.

How to Compile & Run: 

The project is available on GitHub: https://github.com/tancaotrannn/CSE3311 

Prerequisites: Node.js and Git must be installed. 

Clone the repository: 

git clone [https://github.com/tancaotrannn/CSE3311.git](https://github.com/tancaotrannn/CSE3311.git) cd CSE3311 

Install Dependencies: 

npm install recharts

Set up environment variables: 

Create a file named .env.local in the root of the project 

Add Supabase and Spotify API keys to this file.  

Run the development server: 

Npm run dev 

Open your browser and navigate to http://localhost:3000
